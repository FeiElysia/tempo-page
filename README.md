# Tempo: Smart Compressors for Long Video Understanding

[![Project Page](https://img.shields.io/badge/Project-Page-green.svg)](https://feielysia.github.io/tempo-page/)
[![Paper](https://img.shields.io/badge/arXiv-2604.08120-b31b1b.svg?logo=arxiv)](https://arxiv.org/abs/2604.08120)

This is the repository that contains the source code for the **[Tempo project website](https://feielysia.github.io/tempo-page/)**. 

Tempo achieves state-of-the-art long video understanding with a compact 6B model through intent-driven token sparsity.
